# 技術のはなし

- フロントエンドは Next.js をベースに作成し、GitHubActions でビルドして GitHubPages で公開しています
- WASM は Rust 経由で生成しています
- 最適解問題を解くアルゴリズムとしては焼きなまし法を用いています
